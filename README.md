# GODO_DASH
A Endless runner game made in godot that is inspired by GEOMETRY DASH but introduces what if there was no level ends in geometry dash
Aim of this project is to make an expreince like geometry dash but something that is endlesss and randomized not pre-planed like geometry dash

## THEME
EVEN THROUGH ITS A GAME ITS ENDLESS RUNNER SO ENDLESS IS THEME

## TEACH USED
GODOT AND GD SCRIPT

## FEATURES 
### currently added
A player [cube] with jumping and dual jumping animation

A endless ground generator that genrates and delete floor

A randomp spike generator that generate random groups of spike [ 3-6 spikes in a group]

A high score and score shower using labels

A death count showing number of times u died


### to be added
A option menu in which u can choose different playable characters change bg sound etc

A difficulty raise with increase to score

triple jump

super jump

gravity flip

spike generation on roof

collectibles
